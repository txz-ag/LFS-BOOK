1.3. Changelog

This is version 8.4-systemd of the Linux From Scratch book, dated 2019 年 3 月 1 日. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via http://www.linuxfromscratch.org/mirrors.html.
